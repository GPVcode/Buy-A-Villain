Project: Order A Villain

Priorities

  • Building themes on Shopify
  • Customizing themes on Shopify
  • New Findings
  • Set up store's theme architecture. Theme architecture is made up of the components below:
  • Layouts are the base of a theme.
  • Figure out how to set up layout for header and footer which will be reused across the website
  • Templates represent a niche of content displayed in your Shopify store. Such template types include product pages, collection pages, shoping cart, blog posts, etc.
  • Figure out how to create a template type.
  • Reusable modules of content displayed on the webpage. For example, "Image with text" section showcasing an image and text side-by-side with options to customize its placement, colors, etc.
  • Types of content modules that can be added, removed, and reordered WITHIN a section. A block can be an image, video, or text. Blocks are limited to 16 blocks per section.

New Findings

  • 3 reasons merchants consider theme developers:

  • Current theme is outdated or doesn't match the brand's requirements.
  • Current theme is a deeply customized template and slows down the overall website performance.
  • The theme isn't optimized for conversions.

  • Concerns

  • Customized pre-made themeswill be left out of future shopify updates because shopify fails in providing automated ways to upgrade themes while also maintaining customizations
